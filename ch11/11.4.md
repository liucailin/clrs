## 11.4-2

```cpp
HASH-DELETE(T, k)
    i=0
    repeat
        j=h(k,i)
        if T[j]==k
            T[j] = DELETE
        else
            i=i+1
    until i == m
    error "hash table overflow"

HASH-INSERT(T, k)
    i=0
    repeat
        j=h(k,i)
        if T[j]==NIL or T[j]==DELETE
            T[j].k = k
        else
            i=i+1
    until i == m
    error "hash table overflow"
```
